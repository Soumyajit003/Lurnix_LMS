# 📘 Lurnix – Modern LMS Platform

## 🚀 Overview

**Lurnix** is a full-stack Learning Management System (LMS) designed to provide a modern, scalable, and community-driven online learning experience.

It combines:

- 🎓 Course Management  
- 👨‍🏫 Educator Dashboard  
- 👩‍🎓 Student Enrollment System  
- 💬 Community Discussion Platform  
- 🤖 AI Integration (Gemini API)  
- 🔐 Secure Authentication with Clerk  
- 🌙 Modern Dark UI with Glassmorphism  

The platform is built with scalability, modular architecture, and clean UI principles in mind.
